# AI Earth Cheap and Genius

This repository contains the full contents of the **AI Earth Rebuild** workspace, including research loops, experiments, and discovery round notes.

## Repository Structure

- **`ai-earth-rebuild/`**: The core project directory.
  - **`core/`**: Python modules and resource managers (e.g., `resource_manager.py`).
  - **`experiments/`**: Python scripts for running the experiments.
  - **`runs/`**: Execution logs, task metrics, and report outputs for each experiment.
  - **`config/`**: Example configurations and resources.
  - **Research & Discovery Notes**: Markdown files covering foundational models, audit reports, and discovery rounds (Rounds 1–47).

## Main Documents

Some of the key documentation files inside `ai-earth-rebuild/` include:
- [CURRENT_FOCUS.md](ai-earth-rebuild/CURRENT_FOCUS.md) - Current development focus and objectives.
- [VISION_MAP.md](ai-earth-rebuild/VISION_MAP.md) - Overall project vision and roadmap.
- [CORE_ASSUMPTIONS_AUDIT.md](ai-earth-rebuild/CORE_ASSUMPTIONS_AUDIT.md) - Analysis and verification of core project assumptions.
- [PROJECT_IDEA_DISCOVERY_PROTOCOL.md](ai-earth-rebuild/PROJECT_IDEA_DISCOVERY_PROTOCOL.md) - Methodology for project idea discovery.
